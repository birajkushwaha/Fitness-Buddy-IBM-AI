# 🏋️ Fitness Buddy AI

An AI-powered Fitness Assistant built using **IBM BOB**, **IBM watsonx.ai Runtime**, **Meta Llama 3.3 70B Instruct**, and **Python Flask**. The application provides personalized workout plans, nutrition guidance, BMI and calorie calculations, hydration tracking, and AI-powered fitness coaching through a modern web interface.

---

## 📌 Project Overview

Fitness Buddy AI is designed to help users achieve their fitness goals through intelligent and personalized recommendations. The application leverages IBM watsonx.ai foundation models to generate customized workout routines, meal suggestions, and healthy lifestyle advice based on user preferences.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Fitness-Buddy-AI.git
cd Fitness-Buddy-AI
```

### Create Virtual Environment

```bash
python -m venv .venv
```

Activate the environment

Windows

```bash
.venv\Scripts\activate
```

Linux / macOS

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file using `.env.example`.

Example:

```env
IBM_API_KEY=YOUR_API_KEY
WATSONX_PROJECT_ID=YOUR_PROJECT_ID
WATSONX_URL=https://au-syd.ml.cloud.ibm.com
FLASK_SECRET_KEY=YOUR_SECRET_KEY
FLASK_ENV=development
```

### Run the Application

```bash
python app.py
```

Open

```
http://localhost:5000
```
